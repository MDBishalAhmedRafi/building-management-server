# Building Management System - Server

This is the **server-side** of the Building Management System (BMS).  
It provides secure APIs for authentication, apartment management, agreement requests, announcements, coupons, and payments.

---

## 🚀 Features
- 🔐 **JWT Authentication & Role-based Access**
- 🏢 **Apartment Management APIs**
- 📜 **Agreement Request Handling**
- 💳 **Stripe Payment Integration**
- 🎟️ **Coupon Management APIs**
- 📢 **Announcements Management**
- 📂 **MongoDB Collections:** Users, Apartments, Agreements, Payments, Coupons, Announcements

---

## 🛠️ Technologies Used
- **Node.js** + **Express**
- **MongoDB** (Native Driver, not Mongoose)
- **JWT** for authentication
- **Stripe** for payments
- **dotenv** for environment management
- **CORS** + **Helmet** for security
